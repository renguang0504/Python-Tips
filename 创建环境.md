# 创建环境

## 1.修改默认路径

首先手动新建一个文件夹，例如 D:\AnacondaEnvs

```
conda config --add envs_dirs D:\AnacondaEnvs
```



输入以下命令查看当前的配置：

```
conda info
```

在输出的信息中找到 **envs directories**，确保 **D:\AnacondaEnvs** 排在列表的第一个。



## 2.创建环境

```
conda create -n test python=3.9
# -n 后面跟的是你这个环境的名字
# python=3.9 制定你的环境的python版本
```



## 3.激活环境

```
conda activate test
```
